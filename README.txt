# HospitalRegistrationMVC

This is an MVC application built using jsp, servlets and MySQL database(JDBC). This has login functinality. Add new patient and view recently 
registered patients(recent 100 would be displayed in a dynamic table). This application also has search and update functionalities.

jsp is acting as view and servlet as controller and MySql database as my Model. PatientdbUtil.java class in the helper class that communicates
with servelt and database.
